Projeto Cadastro de veiculo

Este projeto consiste em um formulário de cadastro para anúncios de venda de veículos. Seu objetivo é coletar de forma estruturada todas as informações necessárias—como preço, marca, modelo, quilometragem, data de compra, tipo de câmbio e lista de opcionais (Airbag, Ar Condicionado, etc.)—e permitir o upload de imagens, a fim de facilitar e agilizar a criação de um anúncio detalhado para a venda de um carro.
